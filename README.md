# Profile Landing Page

Single-page profile landing page with:

- Hero section and call-to-action buttons
- Skills showcase with animated proficiency meters
- Style mode switch (Copper, Ocean, Mono)
- Responsive layout for desktop and mobile

## Development (Vite)

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
